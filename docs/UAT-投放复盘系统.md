> 模块代号：SP8-AdReview

---

## 1. 验收总览

### 1.1 验收目标

验证投放复盘系统的所有功能模块按照 PRD 要求正常运行，用户可以完成完整的投放复盘工作流。

### 1.2 验收环境

| 项目 | 要求 |
|------|------|
| 运行环境 | Docker Compose 全量部署 |
| 依赖服务 | PostgreSQL、Redis、AI Provider Hub(SP3)、Knowledge Engine(SP4)、Video Analysis(SP6) |
| 前端 | Next.js dev server 或 build 后的静态页面 |
| AI 模型 | 至少配置一个可用的 LLM（Gemini / OpenAI / Anthropic） |
| 测试数据 | 准备千川导出的CSV样例文件、测试视频（已上传SP6分析） |

### 1.3 验收前置条件

- [ ] 所有 Docker 容器正常运行（`docker compose ps` 显示所有服务 healthy）
- [ ] AI Provider Hub 至少配置了一个有效的 API Key
- [ ] Knowledge Engine 正常运行，可创建知识库
- [ ] Video Analysis 中已有至少2条分析完成的视频
- [ ] 准备一份千川导出的 CSV 文件（至少3条素材数据）
