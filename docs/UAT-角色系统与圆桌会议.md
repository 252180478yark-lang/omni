> 模块代号：SP9-Persona  
> 版本：v1.0  
> 日期：2026-04-07

---

## 1. 验收总览

### 1.1 验收目标

验证角色系统与圆桌会议的所有功能模块按照 PRD 要求正常运行，用户可以完成角色切换问答和多角色圆桌讨论的完整工作流。

### 1.2 验收环境

| 项目 | 要求 |
|------|------|
| 运行环境 | Docker Compose 全量部署 |
| 依赖服务 | AI Provider Hub(SP3)、Knowledge Engine(SP4) |
| 前端 | Next.js dev server 或 build 后的静态页面 |
| AI 模型 | 至少配置一个可用的 LLM（Gemini / OpenAI / Anthropic） |
| 测试数据 | 至少一个知识库中已有文档内容 |

### 1.3 验收前置条件

- [ ] 所有 Docker 容器正常运行
- [ ] AI Provider Hub 至少配置了一个有效的 API Key
- [ ] Knowledge Engine 正常运行，已有至少一个包含文档的知识库
- [ ] 浏览器支持 PDF 下载（Chrome / Edge / Firefox）
